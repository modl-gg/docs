## Prerequisites

Before installing modl, make sure you have the following:

- **Java 8 or higher** installed on your server
- A **Minecraft server** running one of the supported platforms:
  - Spigot (or most Spigot forks such as Paper)
  - BungeeCord (or most BungeeCord forks such as Waterfall)
  - Velocity
- A **modl.gg account** with a registered server

## Setup Steps

Getting modl up and running involves four steps:

1. **Register your server** on [modl.gg](https://modl.gg) and complete email verification.
2. **Install the plugin JAR** on your Minecraft server.
3. **Access the web panel** to manage your server, staff, and punishments.
4. **Configure your API key** so the plugin can communicate with the modl platform.

Follow the guides below to complete each step.
