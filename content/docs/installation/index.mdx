## Prerequisites

Before installing modl.gg, make sure you have the following:

- **Java 8 or higher** installed on your server
- A **Minecraft server** running one of the supported platforms:
  - Spigot (or most Spigot forks such as Paper)
  - BungeeCord (or most BungeeCord forks such as Waterfall)
  - Velocity
  - Fabric
- Your Minecraft server must be connected to the internet.

## Setup Steps

Getting modl.gg up and running involves four steps:

0. **For Fabric Installations ONLY:** You must install
 [PacketEvents](https://github.com/retrooper/packetevents/releases) seperately. On all other platforms this
 is handled automatically and you do not need to manually install PacketEvents.
1. **Download the latest version** of the Minecraft plugin at https://github.com/modl-gg/minecraft/releases
2. **Install the plugin** on all your Spigot servers and your Velocity/Bungeecord proxy (or just Spigot server if you aren't using a proxy).
3. **Run your server** and wait for the setup wizard to show up in console. 
4. **Follow the steps** in the setup wizard
 - When installing in bridge-mode (spigot server behind proxy), you can grab the api-key from proxy plugin's boot.yml
 - If you are using Pterodactyl and running a proxy, you will need to allocate port 25590 (or whatever TCP query port
  you specified) to the proxy in order for the backend servers to connect.
5. **Done!** Enjoy the plugin and [join our discord](https://modl.gg/discord) if you need any assistance or want to suggest features.
